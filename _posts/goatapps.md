Here is a list of intentionally vulnerable applications by design to make the learning curve easy for information security enthusiasts. These projects are maintained by OWASP and independent researchers and developers. They call these projects “Goat Projects”.

<b>Web Application

OWASP Web Goat Java : https://owasp.org/www-project-webgoat/
OWASP Web Goat .net : https://github.com/jerryhoff/WebGoat.NET/
OWASP Node.js Goat : https://owasp.org/www-project-node.js-goat/

<b>Mobile Application

Mobile (Android): https://github.com/nvisium-jack-mannino/OWASP-GoatDroid-Project
Mobile (iOS) Swift : https://github.com/OWASP/iGoat-Swift

<b>Internet of Things (IoT)

IoT Goat : https://github.com/OWASP/IoTGoat

<b>Cloud Infrastructure

AWS Cloud Goat : https://rhinosecuritylabs.com/aws/cloudgoat-vulnerable-design-aws-environment/
AWS Goat : https://github.com/ine-labs/AWSGoat
Serverless Goat : https://github.com/OWASP/Serverless-Goat
Azure Goat https://github.com/ine-labs/AzureGoat

<b>Kubernets

Kubernets Goat : https://github.com/ksoclabs/kube-goat
Kubernets Goat : https://github.com/madhuakula/kubernetes-goat

<b>OS Related

SE Linux Goat : https://neutronstar.org/goatselinux.html

<b>CI/CD Goat

CI/CD Goat : https://github.com/cider-security-research/cicd-goat
