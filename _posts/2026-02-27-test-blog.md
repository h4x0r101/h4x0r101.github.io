Here is a list of intentionally vulnerable applications by design to make the learning curve easy for information security enthusiasts. These projects are maintained by OWASP and independent researchers and developers. They call these projects “Goat Projects”.

<b>Web Application

OWASP Web Goat Java : https://owasp.org/www-project-webgoat/<br>
OWASP Web Goat .net : https://github.com/jerryhoff/WebGoat.NET/<br>
OWASP Node.js Goat : https://owasp.org/www-project-node.js-goat/<br>

Mobile (Android): https://github.com/nvisium-jack-mannino/OWASP-GoatDroid-Project<br>
Mobile (iOS) Swift : https://github.com/OWASP/iGoat-Swift<br>

<b>Internet of Things (IoT)

IoT Goat : https://github.com/OWASP/IoTGoat<br>
